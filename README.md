# Interview Prep

A structured collection of advanced study courses for senior full-stack engineering interviews. Covers LLMs, React, Python/FastAPI, infrastructure, Go, and general interview techniques.

---

## Courses

| Course | Description | Status |
|--------|-------------|--------|
| [LLMs](courses/llms/) | LLM engineering + Python — transformers, prompt engineering, RAG, agents, production deployment | Content complete |
| [React](courses/react/) | Advanced React — hooks deep dive, custom hooks, performance, patterns, state management | In progress |
| [Python & FastAPI](courses/python-fastapi/) | FastAPI, Pydantic, async Python, SQLAlchemy, testing, production deployment | Scaffolded |
| [Infrastructure & Scaling](courses/infrastructure/) | System design, databases at scale, caching, Kubernetes, CI/CD, observability | Scaffolded |
| [Golang](courses/golang/) | Go for TypeScript engineers — concurrency, stdlib, HTTP services, production patterns | Scaffolded |
| [General Interview](courses/general-interview/) | Behavioral prep, technical communication, negotiation, coding patterns | Scaffolded |

**Status key**: *Content complete* = study materials written, *In progress* = partially built, *Scaffolded* = detailed outline ready for content

---

## Getting Started

Pick the course most relevant to your upcoming interviews:

- **Targeting AI/LLM roles** → Start with [LLMs](courses/llms/) (has a fast-track 8-12 hour option)
- **Frontend/full-stack roles** → Start with [React](courses/react/)
- **Backend/platform roles** → Start with [Python & FastAPI](courses/python-fastapi/) or [Golang](courses/golang/)
- **Staff+ level interviews** → Start with [Infrastructure & Scaling](courses/infrastructure/) and [General Interview](courses/general-interview/)
